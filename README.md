# 💳 Credit Card Transaction Analysis Dashboard

## 📌 Project Overview
This project is a Power BI dashboard designed to analyze credit card transaction data and provide insights into spending behavior, merchant performance, and financial trends.

---

## 🚀 Features
- 💰 Total Transaction Amount Analysis (73.89M)
- 📊 Total Transactions Tracking (197K)
- 📉 Average Transaction Value
- 🏬 Department-wise Spending Analysis
- 🛍️ Top Merchants by Spending
- 📅 Monthly Spending Trends
- 📌 Category-wise Spending Distribution

## 📊 Key Insights
- Majority spending comes from **Retail category (~76%)**  
- Top merchant contributes significantly higher than others  
- Certain departments dominate transaction volume  
- Spending trends show fluctuations across months

## 🎯 Purpose
- Analyze financial transaction data  
- Identify spending patterns  
- Support data-driven financial decisions
